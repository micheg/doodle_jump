# doodle_jump

Doodle Jump game with Kaboom

### kaboom game project
## setup project
### init with:

    curl -s https://raw.githubusercontent.com/micheg/node_env/main/node_env | bash -s env 14.15.0

### then:

    . env/bin/activate

### install deps:

    npm install

### build:

    npm run build

### watch (automatic rebuild on save):

    npm run watch
### serve:

    npm run serve

### see:

    going to http://0.0.0.0:9999/
